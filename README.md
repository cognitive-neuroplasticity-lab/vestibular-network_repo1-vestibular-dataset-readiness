# Vestibular Dataset Readiness

A research-readiness project evaluating an open normative neuroimaging dataset for future vestibular-network, spatial cognition, and neuroplasticity analyses. This repository focuses on dataset suitability, conceptual ROI definition, feasibility, and strategic preparation for later hypothesis-driven work.

---

## Project Rationale

Vestibular function is represented across distributed cortical and subcortical systems rather than a single isolated brain region. Because of this, meaningful vestibular-network research can begin by operationalizing candidate regions within high-quality open neuroimaging datasets.

This repository was created to establish a rigorous starting point for future work on intrinsic network variability and spatial cognition.

---

## Dataset Used

* **Dataset:** ID1000
* **Collection:** AOMIC
* **OpenNeuro ID:** ds003097
* **Population:** Healthy adults
* **Accessibility:** Public
* **Modalities:** Structural MRI, Diffusion MRI, Resting-state fMRI, Task fMRI, Behavioural Measures

---

## Why This Dataset Was Chosen

This dataset is not explicitly a vestibular-clinic dataset. It was selected because it provides the practical ingredients needed for readiness-stage research:

* Open accessibility for reproducible workflows
* Resting-state fMRI for intrinsic connectivity analyses
* Healthy participants for individual-differences questions
* Multimodal imaging for future expansion
* Behavioural measures for brain-behaviour modelling
* Strong feasibility for literature-derived vestibular ROI analyses

Vestibular systems are treated here as a distributed network construct rather than a single labelled modality.

---

## Candidate Vestibular Regions

The repository uses literature-grounded candidate regions, including:

* Temporo-parietal junction (TPJ)
* Posterior insula
* Inferior parietal lobule
* Precuneus
* Hippocampus
* Parahippocampal cortex
* Cerebellar vestibular zones
